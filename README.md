### 📒 어플리케이션 기능
- **회원 관리**
	- 로그인 및 회원가입
	- 서비스 제공을 위한 회원 등록
	
- **혈당 관리**
	- CGM 데이터 확인 그래프 제공 
	- 머신러닝으로 예측된 저혈당까지 남은 시간 표시
	
- **저혈당 관리**
	- 친구 등록 및 저혈당 알림
  - 저혈당이 발생한 환자의 이메일과 현재 위치 메시지 발송

- **일정 관리**
	- 인슐린, 식사량, 기타 기록
  - 일정 추가 및 삭제 기능
  
- **위급 상황 대응**
	- 현재 위치 기반의 시설 위치 제공
  - 현재 사용자 위치 확인
